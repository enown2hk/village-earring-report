# 村支书耳环事件舆情分析报告

This folder contains the static GitHub Pages site for the "村支书耳环" public opinion report.

- `index.html`: page structure
- `styles.css`: visual system and responsive layout
- `app.js`: data rendering and ECharts setup
- `data.json`: extracted report data
- `assets/`: report chart image assets
- `vendor/`: local ECharts and Lucide bundles
